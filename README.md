A simple Snake Game developed in Java where the snake eats apples, grows in size, and the game ends when it collides with the wall or its own body.

Features:
-Control snake using keyboard arrow keys
-Snake grows in length after eating an apple
-Random apple generation
-Collision detection with walls and snake body
-Game Over screen display

Technologies:
-Java
-Object-Oriented Programming (OOP)
-Java Swing (for GUI)

How to Run:
1.Download or clone the repository
2.Open the project in VS Code / Eclipse
3.Run Main.java
4.Play using arrow keys
